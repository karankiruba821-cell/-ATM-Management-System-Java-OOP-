# 🏧 ATM Management System (Java OOP)

## 📌 Project Overview

This project is a **simple ATM Management System** built using **Java Object-Oriented Programming (OOP)** concepts.

The program simulates basic ATM operations such as checking balance, depositing money, and withdrawing money through a menu-driven console application.

It demonstrates advanced OOP concepts including **Interfaces, Abstract Classes, Inheritance, and Encapsulation**.

---

## 🚀 Features

* Display student account information
* Check account balance
* Deposit money into the account
* Withdraw money from the account
* Menu-driven console interface
* Input handling using Scanner

---

## 🛠️ Technologies Used

* **Java**
* **Object-Oriented Programming (OOP) Concepts**

## ▶️ How to Run the Program

1. Clone the repository.
2. Open the project in a Java IDE or terminal.
3. Compile the program:

```bash
javac Main.java
```

4. Run the program:

```bash
java Main
```

---

## 🧠 OOP Concepts Used

This project demonstrates the following Object-Oriented Programming concepts:

* **Interface** – Defines ATM operations
* **Abstract Class** – Base account structure
* **Inheritance** – StudentAccount extends Account
* **Encapsulation** – Private variables with getters/setters
* **Polymorphism** – Interface method implementation

---

## 📋 Program Workflow

1. The program creates a student account with initial balance.
2. A menu is displayed with options:

   * Display Info
   * Check Balance
   * Deposit
   * Withdraw
   * Exit
3. The user selects an option.
4. The program performs the selected operation.
